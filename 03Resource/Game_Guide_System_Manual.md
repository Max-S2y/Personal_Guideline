# 🎮 游戏攻略系统：全流程操作指南

> [!abstract] 系统核心逻辑
> 本系统采用 **“项目文件中心化 + 日记碎片化记录”** 的模式。
> 1. **中心化**：每个游戏必须在 `01Project` 文件夹下有一个专属的项目笔记（如 `[[鬼泣5]]`）。
> 2. **碎片化**：你不需要每天打开项目笔记写日记，只需要在 **每日日记 (Daily Note)** 中提及该游戏链接并打上标签（如 `[[鬼泣5]] #进行中`）。
> 3. **自动化**：`01Project` 主页会自动扫描你的日记，更新游戏的当前状态和最新进度。

---

## 🚀 快速上手流程

### 第一步：创建新游戏档案
1. 进入 [[01Project]] 主页。
2. 点击顶部的 **“➕ 新建游戏项目”** 按钮。
3. 输入游戏名称（例如 `黑神话：悟空`）。
4. 系统会自动创建文件 `01Project/黑神话：悟空.md` 并应用模板。

### 第二步：在日记中记录进度
每天写日记时 (`Ctrl+P` -> `Open today's daily note`)，在 `## 🎮 今日进度` 章节下记录：

> - [[黑神话：悟空]] 今天打过了虎先锋，太难了 #进行中
> - [[赛博朋克2077]] 这里有个bug #挂机

**关键点：**
*   必须使用 **双括号** `[[ ]]` 链接到游戏项目。
*   必须打上 **状态标签**（见下文）。

### 第三步：查看自动看板
回到 [[01Project]] 主页，你会发现：
*   `黑神话：悟空` 自动出现在了 **🔥 正在攻略** 列表中。
*   列表右侧显示了你刚才在日记里写的：“今天打过了虎先锋...”。

## 🏷️ 状态标签字典

| 标签 | 含义 | 归类列表 | 适用场景 |
| :--- | :--- | :--- | :--- |
| `#新开` | 刚开始玩 | 🔥 正在攻略 | 创建档案后的第一第一次游玩 |
| `#进行中` | 正在游玩 | 🔥 正在攻略 | 日常推进度 |
| `#待玩` | 想玩但没空 | 📦 待玩清单 | 买了游戏占坑，或者加入愿望单 |
| `#通关` | 主线完成 | 🏆 已完成 | 看到制作人员名单 |
| `#全成就` | 完美达成 | 🏆 已完成 | 拿到白金奖杯/全成就 |
| `#弃坑` | 不想玩了 | 🗑️ 已放弃 | 游戏不好玩或太难 |

## ❓ 常见问题 (FAQ)

### Q: 我在日记里写了，但主页没显示？
1. **检查链接**：确认你写的是 `[[游戏名]]` 变蓝色的链接，而不是纯文本。
2. **检查文件**：确认 `01Project` 文件夹里确实创建了这个游戏的文件。
3. **检查标签**：确认标签拼写正确（如 `#进行中`），且标签前后有空格。

### Q: 怎么把游戏移动到“已完成”？
不需要移动文件。只要你在今天的日记里写上一句：
> `- [[某游戏]] 终于通关了！ #通关`

`01Project` 主页下次刷新时，就会自动把它从“正在攻略”移动到“已完成”区域。

### Q: 什么是“Game Project Template”？
这是系统内置的模版文件，位于 `03Resource/templates/Game Project Template.md`。它定义了新游戏项目的结构（封面、进度条、攻略区）。你可以根据自己喜好修改它。
