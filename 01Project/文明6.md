## 📝 攻略笔记
> [!TIP]
> 在每日日记中引用此游戏并打上进度标签，即可自动更新状态：
> - `#新开` : 刚刚开始
> - `#进行中` : 正在攻略
> - `#待玩` : 加入积压列表
> - `#通关` : 主线完成
> - `#全成就` : 完美达成

```dataviewjs
// 自动获取关联的每日日记日志
// 获取当前文件信息
const currentPath = dv.current().file.path;
const currentName = dv.current().file.name;

// 查找日记中提及此游戏的列表项
const logs = dv.pages('"Diary"').file.lists
    .where(l => {
        // 方法1：检查是否包含指向当前文件的链接 (最准确)
        const linked = l.outlinks.some(o => o.path === currentPath || o.path.endsWith(currentPath));
        // 方法2：如果没链接，检查文本是否包含 [[文件名]] (兜底)
        const textMatch = l.text.includes(`[[${currentName}]]`);
        return linked || textMatch;
    })
    .sort(l => l.link.file.day, 'desc');

if (logs.length > 0) {
    dv.header(3, "📅 游玩日志回顾");
    dv.table(["日期", "日志内容", "状态标签"], 
        logs.map(l => [
            l.link, 
            l.text.replace(/\[\[.*?\]\]/g, "").replace(/#\S+/g, "").trim(), // 清理链接和标签只显示内容
            l.tags.join(", ")
        ])
    );
} else {
    dv.paragraph(`暂无游玩日志。请在日记中使用 **[[${currentName}]]** 提及此项目。`);
}
```
